# Web xem phim
Web được thiết kế để xem phim ( Sử dụng KKPhim API để lấy các thông tin về phim)
## Mục lục
- [Tính năng](#tính-năng)
- [Kiến trúc](#kiến-trúc)
- [Công nghệ sử dụng](#công-nghệ-sử-dụng)
- [Các bước cài đặt](#các-bước-cài-đặt)
- [Một số hình ảnh web](#một-số-hình-ảnh-web)

## Tính năng
+ Phim bộ, Phim lẻ, Hoạt hình, Phim theo thể loại, Tìm kiếm phim, Bộ lọc phim
## Kiến trúc
Dự án được xây dựng dựa trên Kiến trúc MVC.
## Công nghệ sử dụng
+ Backend : ASP.NET Core MVC .NET 8
+ Frontend : Razor

## Các bước cài đặt
1. Clone Repository
2. Chạy web
